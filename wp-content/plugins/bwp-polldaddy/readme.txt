== Description ==

This plugin helps you add Polldaddy Polls to your WordPress website easily. BWP Polldaddy Polls focuses on the front end rather than the back end, i.e. you edit your polls on polldaddy.com and you choose how to show them on your website.

**Demo**

You can view a working instance of this plugin on the right sidebar of [Better WordPress](http://betterwp.net) homepage.

**Some Features**

* Simple, lightweight and easy to use
* Allow you to update your polls manually or automatically using a configurable time settings
* Offer SSL support for paid Polldaddy accounts
* Display polls using widget
* Display polls using shortcode (`bwp-polldaddy`)
* Display polls using template functions (`bwp_polldaddy`)
* Limit the number of polls to display
* Sort polls based on IDs, creation date, numbers of responses, or sort randomly
* WordPress Multi-site compatible

== Installation ==

1. Upload the `bwp-polldaddy` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the Plugins menu in WordPress. After activation, you should see a menu of this plugin under Settings.
3. Configure the plugin. 
4. Add polls to your site using either widget, shortcode (`bwp-polldaddy`), or template function (`bwp_polldaddy()`).
5. Enjoy!

Visit [Plugin's Official Page](http://betterwp.net/wordpress-plugins/bwp-polldaddy/) for more information!
